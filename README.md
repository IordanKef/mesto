# Проект: Место
## Учебный проект

Страничка о путешествиях пользователя, созданная в ходе курса веб-разработки от [Яндекс.Практикума](https://practicum.yandex.ru/).

**Использованые технологии:**
- HTML
- CSS:
  - Адаптивная вёрстка
  - Медиа-запросы
  - Flex
  - Grid layout
- Javascript:
  - Кнопки
  - Формы
  - Темплейты
  - Попапы
